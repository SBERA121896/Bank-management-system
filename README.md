# Bank managment syatem 
A Python-based banking management system that allows users to create accounts, deposit money, withdraw money, transfer funds, check balance, and close accounts securely.

## Features
- Create New Account
- Deposit & Withdraw Money
- Transfer Funds
- Balance Checking
- Secure PIN Authentication
- Account Deletion
- File Handling
- PIN Hashing using SHA-256

## Technologies Used
- Python
- File Handling
- OOP Concepts
